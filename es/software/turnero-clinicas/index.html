    FAQ

    Lo que una clinica pregunta antes de comprar

    Necesitamos hardware propietario para arrancar?

    No. El piloto esta pensado para trabajar con navegador, una TV o monitor, una PC o tablet y una impresora termica compatible.

    Esto reemplaza el software medico completo de la clinica?

    No en esta etapa. La suite compite como operacion de sala + agenda + recordatorios + reporting, no como EMR completo.

    Se puede empezar por una sola sede?

    Si. El modelo inicial esta pensado para una sede piloto y luego activar branding, sedes lite y dashboard consolidado si el flujo funciona.

    Como bajan el no-show si el paciente no usa kiosco?

    La mejora no depende solo del kiosco. Entra por recordatorios, check-in movil, vista de turno y una recepcion que ve la misma informacion.

    Si recepcion lo adopta rapido, el paciente lo entiende y gerencia lo puede medir, la suite ya tiene historia de compra.

    Empiece por la operacion diaria, pruebe el flujo paciente y cierre con el dashboard de sede cuando llegue la conversacion de presupuesto.
