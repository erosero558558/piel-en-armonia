FAQ

Preguntas habituales antes de contratar

El plan Gratis sirve para una clinica completa?

Sirve para validar el flujo y entrenar la primera operacion. Si ya tiene recepcion, sala y mas de un doctor, normalmente la clinica cae mejor en Pro.

El precio Pro incluye implementacion?

Incluye onboarding guiado para la primera sede y configuracion inicial. Si la clinica necesita rollout formal multi-sede, eso se mueve a Enterprise.

Que cambia en Enterprise?

Cambia gobierno: mas sedes, integraciones, soporte dedicado, whitelabel y despliegue acordado por alcance real.
