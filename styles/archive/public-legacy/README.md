# Public Legacy CSS Archive

Este directorio preserva snapshots y variantes minificadas legacy del CSS
publico que ya no forman parte del front door activo del repo.

Reglas:

- Se conservan solo como referencia historica.
- No forman parte del runtime publico ni admin activo.
- El runtime canonico usa `styles.css`, `styles-deferred.css`,
  `styles-critical.css`, `styles-astro.css`, `styles-telemedicina.css`,
  `admin-v3.css`, `queue-ops.css`, `queue-kiosk.css`, `queue-display.css`,
  `ops-design-system.css` y `legal.css`.

CSS archivado:

- `styles.min.css`
- `styles.optimized.css`
- `styles-critical.min.css`
- `styles-deferred.min.css`
