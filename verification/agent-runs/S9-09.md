# S9-09 — Callback cockpit

## Resumen

- Se actualizo el cockpit de callbacks en admin para recepcion.
- La cola ahora prioriza pendientes antes que contactados y mantiene el orden por score.
- Se agrego filtro de dia en la UI y se expuso ultimo contacto en la tarjeta foco y en cada callback card.
- `LeadOpsService` ahora propaga `scoreSummary` y `scoreFactors` para que el cockpit muestre el contexto del score real.
