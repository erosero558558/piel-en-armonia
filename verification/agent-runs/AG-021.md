# AG-021 - C1 exit gate en CI (staging)

## Goal
Automatizar salida C1 en CI con smoke + Playwright contract de Figo antes de merge.

## Implementado
- `.github/workflows/ci.yml`
  - Nuevo job `c1-exit-gate`
  - Ejecuta contrato Playwright de Figo contra staging
  - Smoke del endpoint de chat con fallback `'/api/chat' -> '/figo-chat.php'`
  - Gate de latencia (p95 Figo <= 3000ms)
  - Integracion con `paths-filter` para skip cuando no hay cambios relevantes
  - `build` depende de `c1-exit-gate`
- `tests/figo-chat-contract.spec.js`
  - Soporte para `TEST_FIGO_CHAT_ENDPOINT_PATH` (default `/figo-chat.php`)

## Validacion local
- Parse YAML:
  - `.github/workflows/ci.yml` -> OK
- JS syntax:
  - `node --check tests/figo-chat-contract.spec.js` -> OK
  - `node --check eslint.config.js` -> OK (config relacionada al gate CI)

## Limite de esta evidencia
El gate C1 no puede validarse end-to-end aqui porque requiere ejecucion remota en GitHub Actions con secrets/entorno staging.
