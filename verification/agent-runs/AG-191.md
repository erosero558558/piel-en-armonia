# AG-191 - Turnero native readme portability

Fecha: 2026-03-11
Owner: Ernesto
Executor: codex

## Objetivo

Dejar la documentacion de `turnero-desktop` y `turnero-sala-tv-android` portable y ejecutable sin rutas absolutas ni ambiguedad de comandos entre entornos.

## Cambios consolidados

- Se reemplazaron referencias absolutas por rutas relativas del repo.
- `turnero-desktop/README.md` ahora incluye comandos portables desde la raiz usando `npm --prefix`.
- `turnero-sala-tv-android/README.md` ahora documenta `gradlew` y `gradlew.bat`, ademas de un ejemplo portable de `assembleRelease` con propiedades inyectadas.
- Ambos README ahora referencian el gate canonico `npm run gate:turnero`.

## Validacion ejecutada

- Revision manual de enlaces relativos y comandos documentados.
- `npm run gate:turnero`

## Resultado

- La documentacion ya no depende de rutas locales tipo `/home/...`.
- Los pasos de desarrollo/build/release quedan claros para Windows, macOS y Linux.
