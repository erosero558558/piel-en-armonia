# S44-01 — Gate de lanzamiento

Fecha: 2026-04-01

## Cambios

- `bin/verify.js` ahora soporta `--gate launch` con 13 checks reales para auth, booking, consentimiento, pagos, documentos, GA4, gobernanza y health.
- `bin/verify.js` ahora también soporta `--json` y `--task <id>` para tooling interno compatible con `bin/mark-done.js`.
- `tests-node/verify-cli.test.js` cubre el contrato del launch gate, la salida JSON del CLI y el resultado por tarea.

## Verificación

```bash
node --check bin/verify.js
node --check tests-node/verify-cli.test.js
node --test tests-node/verify-cli.test.js
node bin/verify.js --gate launch
node bin/verify.js --gate launch --json
node bin/verify.js --task S44-01 --json
```

## Resultado

- `node bin/verify.js --gate launch` pasa con `13/13`.
- El gate deja explícitos los warnings tolerados fuera del contrato (`done-without-evidence` base), sin bloquear el preflight de lanzamiento.
- El JSON del CLI ya expone `withoutEvidence`, `noCheckTasks`, `taskHasRule` y `taskResult` para integraciones de gobernanza.
