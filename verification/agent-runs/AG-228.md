# AG-228 - Migrar queue-kiosk al helper compartido de surfaces

## Summary

- Reemplacé mocks inline repetidos en `tests/queue-kiosk.spec.js` por el helper compartido `tests/helpers/turnero-surface-mocks.js`.
- Alineé los escenarios de branding, degradación por ruta inválida y fallback de `clinic-profile` para que usen el mismo contrato compartido que `queue-display`.
- Verifiqué el consumo del helper compartido con el contrato Node y la suite Playwright del kiosco.

## Verification

```powershell
npx eslint tests/queue-kiosk.spec.js tests-node/turnero-surface-mocks-helper.test.js tests/helpers/turnero-surface-mocks.js
node --test tests-node/turnero-surface-mocks-helper.test.js
$env:TEST_LOCAL_SERVER_PORT='8046'; npx playwright test tests/queue-kiosk.spec.js --workers=1
```

## Outcome

- `18/18` pruebas Playwright del kiosco pasando.
- `6/6` pruebas Node del helper compartido pasando.
- ESLint sin errores; solo quedaron warnings previos de `playwright/prefer-locator` en `tests/queue-kiosk.spec.js`.
