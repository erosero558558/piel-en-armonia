# S4-23 — Package.json audit

Fecha: `2026-03-30`

## Objetivo

Auditar [package.json](/Users/luciaguadalupecaizasanchez/Documents/GitHub/Aurora-Derm/package.json) para detectar scripts que apuntan a archivos archivados o inexistentes y dejar lista accionable para limpieza.

## Comandos usados

```bash
node -e "const p=require('./package.json'); console.log(Object.keys(p.scripts).length)"
rg -n "_archive|archive/|archive:" package.json
npm run turnero:verify:pilot:local
npm run content:public-v4:validate
test -d src/apps/turnero-desktop/dist && echo exists || echo missing
```

## Hallazgos

- `package.json` contiene `260` scripts.
- No hay referencias directas a `_archive/` ni a rutas archivadas inválidas.
- Se detectaron `21` scripts rotos por `19` referencias inexistentes, agrupados en:
  `Public V5/V6 Sony`, `Admin rollout`, `Governanza legacy`, `Validadores public-v4/public-v5` y `Turnero release/contracts`.
- `turnero:stage:pilot:local` depende de `src/apps/turnero-desktop/dist`, ausente en checkout limpio; se clasificó como prerequisito de build, no como huérfano.

## Reproducción confirmada

- `npm run turnero:verify:pilot:local` falla con `MODULE_NOT_FOUND` para `bin/verify-turnero-release-bundle.js`.
- `npm run content:public-v4:validate` falla con `MODULE_NOT_FOUND` para `bin/validate-public-v4-catalog.js`.
