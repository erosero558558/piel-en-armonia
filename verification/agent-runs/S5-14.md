# S5-14 — WhatsApp bot IA fuera de horario

## Cambios

- `WhatsappOpenclawPlannerClient` ahora resuelve localmente dos casos deterministas:
  - FAQ de fuera de horario sobre horarios, ubicacion y que llevar a la cita.
  - preguntas clinicas que deben escalarse a seguimiento humano.
- `WhatsappOpenclawConversationOrchestrator` ahora maneja `handoff_clinical`, deja nota en el draft y marca la conversacion en `human_followup`.
- Se agrego cobertura de integracion para ambos casos en `WhatsappOpenclawControllerTest`.

## Verificacion

```bash
php -l lib/whatsapp_openclaw/PlannerClient.php
php -l lib/whatsapp_openclaw/ConversationOrchestrator.php
php -l tests/Integration/WhatsappOpenclawControllerTest.php
git diff --check -- lib/whatsapp_openclaw/PlannerClient.php lib/whatsapp_openclaw/ConversationOrchestrator.php tests/Integration/WhatsappOpenclawControllerTest.php
php vendor/bin/phpunit --no-coverage tests/Integration/WhatsappOpenclawControllerTest.php
node bin/gate.js S5-14
```

## Resultado

- El bot responde informacion util aun cuando el mensaje entra fuera de horario.
- La respuesta fuera de horario cubre horarios, direccion de la clinica y que llevar a la cita.
- Las preguntas clinicas ya no intentan resolverse automaticamante: quedan escaladas a humano con nota trazable en el draft.
