# AG-183 - Operator auth backend with OpenClaw ChatGPT

## Scope

- Added backend operator auth flow with signed challenge completion for OpenClaw/ChatGPT.
- Kept legacy `admin-auth.php?action=login|login-2fa` available when `PIELARMONIA_OPERATOR_AUTH_MODE` is not enabled.
- Exposed canonical API resources:
    - `POST /api.php?resource=operator-auth-start`
    - `GET /api.php?resource=operator-auth-status`
    - `POST /api.php?resource=operator-auth-complete`
    - `POST /api.php?resource=operator-auth-logout`
