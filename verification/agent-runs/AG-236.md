# AG-236 - Alinear readiness health del piloto web en admin queue spec

## Resumen

- Se alineó `tests/admin-queue.spec.js` con el contrato vigente del hub de
  piloto web.
- El caso donde `/health` reporta otra clínica activa ya no bloquea el piloto:
  hoy la UI lo presenta como warning operacional con el título
  `Piloto web casi listo`.
- El test quedó renombrado para reflejar esa semántica nueva sin perder la
  verificación del detalle `clinica-sur-real` en readiness/issues.
