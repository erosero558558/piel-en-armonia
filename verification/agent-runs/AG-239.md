# AG-239 - Expandir la barra de comandos admin hacia el frente clínico y OpenClaw

## Resumen

- Se extendió `parseQuickCommand(...)` para enrutar aliases clínicos como
  `historia clínica`, `telemedicina`, `intake`, `paciente` y `caso` hacia la
  sección `clinical-history`, sin romper el carril existente de `OpenClaw` /
  `copiloto`.
- La paleta de comandos ahora insinúa estos aliases desde el placeholder para
  que la navegación rápida exponga el frente clínico como primera clase.
- Se alinearon los listeners de `shell` e `input` para que ejecutar un comando
  cierre la paleta, limpie el input y abra el panel operativo correcto tanto
  para `Historia clínica` como para el copiloto.
- Se actualizó el contrato DOM del shell para reflejar el surface canónico del
  panel del agente y sus `data-action`.
- Se agregó cobertura Node y Playwright para fijar tanto el contrato de routing
  como la navegación real desde `Ctrl+K`.
