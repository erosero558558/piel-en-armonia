# AG-235 - Stabilize admin quick-nav runtime-ready shortcuts

- Fecha: 2026-03-14
- Owner: Ernesto
- Executor: codex

## Resumen

Se estabilizó `tests/admin-quick-nav.spec.js` para los atajos que dependen del
runtime principal del admin (`Ctrl+K` y `Alt+Shift+I`).

El problema no estaba en los shortcuts sino en una carrera de arranque: el spec
disparaba el teclado antes de que `admin.js` terminara de marcar
`data-admin-ready="true"` y de enganchar los listeners globales del bundle.

Se agregó un helper `waitForAdminRuntimeReady(page)` y se reutilizó antes de los
casos que ejercen la barra de comandos y el copiloto.
