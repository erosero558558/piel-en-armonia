# AG-243 - Admin queue pilot cut: quick-nav y hub piloto en admin v3

Fecha: 2026-03-14
Owner: codex_frontend
Dominio: queue

## Objetivo

Abrir el primer corte demostrable del foco `FOCUS-2026-03-admin-operativo-cut-1` haciendo visible un quick-nav superior en admin v3 y reforzando el hub piloto de `queueOpsPilot` dentro del frente operativo.

## Cambio aplicado

- Se inyecto un quick-nav superior en `admin v3` desde `src/apps/admin-v3/ui/frame/shell.js` para priorizar `Inicio`, `Pendientes` y `Turnero`, reutilizando el runtime de navegacion ya existente por `data-section`.
- Se agrego estilo dedicado en `admin-v3.css` para que el quick-nav quede integrado al topbar en desktop y movil.
- Se reforzo el hub piloto en `src/apps/admin-v3/shared/modules/queue/render/section/install-hub/pilot/render.js` con copy visible de `Hub piloto del admin v3`, nota de corte operativo y foco accesible sobre `#queueOpsPilotTitle`.
- Se ajusto `src/apps/admin-v3/shared/modules/queue/render/section/install-hub/pilot/actions.js` para devolver foco al encabezado del hub tras confirmar sugeridos y mantener al operador en el mismo corte.
- Se actualizaron `tests/admin-quick-nav.spec.js` y `tests/admin-v3-visual.spec.js` para cubrir la navegacion superior y la presencia del hub piloto.
- Se recompilo el bundle canónico del admin (`admin.js` + `js/admin-chunks/**`) y se limpio el chunk huérfano con `npm run chunks:admin:prune`.
