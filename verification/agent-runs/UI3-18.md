# UI3-18 — Page loading bar

## Resumen

- Se creó `js/aurora-nprogress.js` como loader liviano sin dependencias, con barra superior de 3px y estado compartido entre navegación saliente y carga entrante.
- Los layouts `BaseLayout`, `PublicShellV3`, `PublicShellV5` y `PublicShellV6` ahora cargan el script compartido.
- Se añadió cobertura de contrato y una spec Playwright sobre una ruta pública real para validar que el loader se inyecta en runtime.

## Archivos

- `js/aurora-nprogress.js`
- `src/apps/astro/src/layouts/BaseLayout.astro`
- `src/apps/astro/src/layouts/PublicShellV3.astro`
- `src/apps/astro/src/layouts/PublicShellV5.astro`
- `src/apps/astro/src/layouts/PublicShellV6.astro`
- `tests-node/public-loading-bar-contract.test.js`
- `tests/public-loading-bar.spec.js`

## Validacion

- `node --check js/aurora-nprogress.js`
- `node --test tests-node/public-loading-bar-contract.test.js`
- `npm run build:public:v6`
- `TEST_LOCAL_SERVER_PORT=8421 npx playwright test tests/public-loading-bar.spec.js --workers=1`
- `git diff --check`

## Resultado

- Las superficies publicas cargan una barra visual fina durante navegacion y primer paint sin depender de NPM.
- El loader queda disponible como `window.__auroraPageLoader` para futuros hooks de navegacion.
