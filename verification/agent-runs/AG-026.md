# AG-026 - Agent Autopilot workflow hardening (local)

## Goal
Mejorar robustez y diagnostico del workflow `agent-autopilot` para fallas de permisos/validacion en CI.

## Implementado
- `.github/workflows/agent-autopilot.yml`
  - Permisos explicitos para pasos que consultan/leen APIs usadas por `intake/reconcile`:
    - `issues: read`
    - `actions: read`
    - `pull-requests: read`
  - Separacion de validaciones en pasos con diagnostico mas claro
  - Snapshot JSON en fallo para facilitar analisis posterior

## Validacion local
- Parse YAML:
  - `.github/workflows/agent-autopilot.yml` -> OK

## Limite de esta evidencia
No se puede validar aqui la ejecucion real del workflow ni permisos efectivos del token de GitHub Actions.
