# S4-24 - CSS dead code audit

Fecha: `2026-03-30`

## Objetivo

Auditar los archivos CSS de la raiz del repo para verificar cuales siguen importados desde superficies web vivas y listar los huerfanos sin borrarlos.

## Comandos usados

```bash
find . -maxdepth 1 -type f -name '*.css' | sort
rg -n "\\.css" -g '*.html' -g '*.php' -g '*.astro' -g '*.js' -g '*.ts' -g '*.tsx' templates src js *.html *.php
for f in admin-v3.css legal.css ops-design-system.css queue-display.css queue-kiosk.css queue-ops.css styles-astro.css styles-critical.css styles-deferred.css styles-telemedicina.css styles.css; do
  rg -n --glob '!node_modules/**' --glob '!.generated/**' --glob '!.codex-worktrees/**' --glob '!tests/**' --glob '!tests-node/**' --glob '!verification/**' --glob '!vendor/**' --glob '!dist/**' --glob '!build/**' -e "$f" templates src js *.html *.php || true
done
```

## Hallazgos

- La raiz contiene `11` CSS.
- `7` siguen vivos: `admin-v3.css`, `queue-display.css`, `queue-kiosk.css`, `queue-ops.css`, `styles-astro.css`, `styles-deferred.css`, `styles.css`.
- `4` quedaron huerfanos: `legal.css`, `ops-design-system.css`, `styles-critical.css`, `styles-telemedicina.css`.
- El bloque huerfano suma `53135 B`.
