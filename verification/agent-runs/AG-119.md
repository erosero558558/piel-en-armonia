# AG-119 - Turnero sala VI: hardening numpad cross-platform windows/mac

Fecha de cierre: 2026-02-27  
Owner: Ernesto  
Executor: codex

## Alcance ejecutado

- Hardening de detección numpad en `src/apps/admin/index.js` con soporte:
  - `KeyboardEvent.code` (`NumpadEnter`, `NumpadAdd`, `NumpadSubtract`, `NumpadDecimal`)
  - fallback por `key + location=3` para teclados con mapeo irregular.
- Compatibilidad extendida para decimal regional:
  - `.` / `,` / `Delete` en layouts Windows/Mac.
- Guardrails mantenidos:
  - `Enter` del teclado principal no llama turnos.
  - eventos con foco en inputs o con modificadores se ignoran.
- Cobertura QA ampliada con caso específico cross-platform en `tests/admin-queue.spec.js`.
