# S7-04 — Rate limiting en auth sensible

## Resumen

- Se agrego emision de `X-RateLimit-Limit`, `X-RateLimit-Remaining` y
  `X-RateLimit-Reset` desde `lib/ratelimit.php`.
- Se alineo `admin-auth.php?action=login` al contrato de `5` intentos por
  minuto por IP, manteniendo el bloqueo adicional para repeticion de fallos.
- Se documento el contrato operativo en `docs/SECURITY.md`.
- Se agrego una prueba HTTP que valida headers y `429` real sobre el endpoint
  de login admin.
