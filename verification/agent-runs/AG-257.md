# AG-257

## Resumen

- acote el soporte de `release-publish` para `frontend-public` a un set minimo de archivos de build, contratos y artefactos canonicos del release publico;
- mantuve el bloqueo conservador para cualquier archivo fuera de ese permiso acotado;
- preserve el lane `frontend_content` al iniciar un `--release-publish` de `frontend-public` ya bien clasificado, evitando que la reinferencia por `files` lo degradara a `backend_ops`.

## Archivos

- `tools/agent-orchestrator/domain/task-guards.js`
- `tools/agent-orchestrator/commands/task.js`
- `bin/validate-agent-governance.php`
- `tests-node/orchestrator/task-guards.test.js`
- `tests-node/agent-orchestrator-cli.test.js`
