# S4-15 — Promociones

## Resumen

Se creó `es/promociones/` como plantilla pública para campañas rotativas de Aurora Derm.

Incluye:

- hero de campaña destacada
- cards para campañas activas y próximas
- filtros funcionales por tipo (`todas`, `consulta`, `procedimientos`, `temporada`)
- CTA directa por WhatsApp para cada campaña
- bloque de reglas para rotar promociones sin rehacer toda la página

## Archivos

- `es/promociones/index.html`
- `es/promociones/promociones.css`

## Validación

Comandos ejecutados:

```bash
git diff --check -- es/promociones/index.html es/promociones/promociones.css
rg -n "Mes de la piel|Día de la Madre|data-promo-filter|data-promo-card" es/promociones/index.html es/promociones/promociones.css
```

Validación de navegador en `http://127.0.0.1:8013/es/promociones/`:

- `title` correcto
- `h1` visible
- filtro `temporada` funcionando con `2` cards visibles

## Resultado

La landing quedó lista como superficie reutilizable para campañas comerciales de temporada y campañas clínicas cortas con salida inmediata a WhatsApp.
