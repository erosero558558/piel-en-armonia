# S16-12 — Release pack de schema para Custom GPT

## Resumen

- Se agrego `x-schema-version` a `openapi-openclaw.yaml` con formato `YYYY-MM-DD-hash`.
- Se creo `bin/gen-gpt-schema-pack.js`, que normaliza la version del schema a partir del hash canonico del YAML y genera `docs/gpt-schema-pack-latest.md`.
- Se enlazo `docs/chatgpt-custom-gpt-instructions.md` al pack vigente para que el equipo medico pueda verificar rapidamente que schema debe importar.

## Archivos

- `openapi-openclaw.yaml`
- `docs/chatgpt-custom-gpt-instructions.md`
- `docs/gpt-schema-pack-latest.md`
