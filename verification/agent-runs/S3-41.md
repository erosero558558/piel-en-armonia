# S3-41 — CIE-10 autocomplete widget

## Resumen

Se agrego `js/cie10-autocomplete.js` al admin para sugerencias CIE-10 en tiempo real sobre los campos clinicos de diagnostico en HCE. El widget hace `debounce` de 200 ms, consulta `GET /api.php?resource=openclaw-cie10-suggest&q=...`, muestra dropdown navegable por teclado y aplica la seleccion sobre el formulario activo.

## Decisiones de compatibilidad

- El campo `clinician_cie10` sigue persistiendo solo el codigo CIE-10 para no romper la serializacion actual de `cie10Sugeridos`.
- La descripcion seleccionada se refleja en el campo clinico acompanante cuando esta vacio y tambien queda visible como resumen inmediato bajo el input (`codigo - descripcion`).
- Se corrigio ademas el boot del admin importando `renderClinicalCompareFlow` en `src/apps/admin-v3/sections/clinical-history/render/index.js`; sin ese import el runtime compilado quedaba en `data-admin-ready=false` y bloqueaba la workspace completa.
