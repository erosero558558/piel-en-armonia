# S4-17 — Gift cards

## Resumen

Se creó la nueva superficie `es/gift-cards/` con:

- montos predefinidos (`$50`, `$100`, `$180`, `$250`)
- personalización para destinatario, remitente y mensaje
- generación de código único tipo `AUR-180-XXXXXXXX`
- PDF descargable generado en navegador
- enlace listo para compartir por WhatsApp

## Archivos

- `es/gift-cards/index.html`
- `es/gift-cards/gift-cards.css`
- `es/gift-cards/gift-cards.js`

## Validación

Comandos ejecutados:

```bash
node --check es/gift-cards/gift-cards.js
git diff --check -- es/gift-cards/index.html es/gift-cards/gift-cards.css es/gift-cards/gift-cards.js
rg -n "data-gift-amount|gift-preview-code|gift-download|Gift card Aurora Derm" es/gift-cards/index.html es/gift-cards/gift-cards.js es/gift-cards/gift-cards.css
```

Validación de navegador en `http://127.0.0.1:8013/es/gift-cards/`:

- carga correcta de la ruta
- selección de monto `$180`
- generación de código en pantalla
- feedback visible de éxito
- descarga real de PDF con nombre:
  - `AUR-180-8RJVAOW1.pdf`

## Resultado

La landing ya permite emitir una gift card simple y usable sin depender de backend extra ni librerías nuevas de PDF.
