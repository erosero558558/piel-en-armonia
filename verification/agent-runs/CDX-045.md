# CDX-045 — Orchestrator gate unblock for CDX-009 closeout

## Objetivo

Desbloquear el gate transversal del orquestador para que el close canónico de `CDX-009` deje de fallar por `publish_gate_failed`, sin tocar el root principal mixed-lane ni relajar checks de publish/close.

## Alcance

- Portar al clon limpio solo los hunks transversales ya útiles del root principal.
- Corregir los fixtures y contratos que todavía dejan `agent-test` en rojo.
- Normalizar la metadata histórica de `AG-251` para alinear el validador PHP con el naming canónico actual de `operator_auth`.
