## Implementation

1. Added a V6 search overlay in the public header with route index data from V6 content.
2. Wired runtime search open, close, live filtering, Escape handling, focus trap, and body scroll lock.
3. Added news strip detail panel markup and runtime toggle state.
4. Replaced dead affordances with working controls while preserving `data-v6-*` contracts.
5. Added explicit hidden-state CSS guards for overlay and expandable panel.
6. Regenerated public static artifacts after syncing Astro output to root.
