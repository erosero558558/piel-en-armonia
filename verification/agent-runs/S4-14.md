# S4-14 — Programa de referidos

## Resumen

Se publicó la nueva landing `es/referidos/` con:

- beneficio claro para quien refiere y para la persona referida
- CTA principal `"Comparte tu link"`
- flujo funcional de compartir con Web Share API y fallback
- bloque de condiciones, pasos y FAQ para reducir fricción comercial

## Archivos

- `es/referidos/index.html`
- `es/referidos/referidos.css`

## Validación

Comandos ejecutados:

```bash
git diff --check -- es/referidos/index.html es/referidos/referidos.css
rg -n "Comparte tu link|data-referral-share|Programa de referidos Aurora Derm" es/referidos/index.html es/referidos/referidos.css
```

Validación de navegador en `http://127.0.0.1:8013/es/referidos/`:

- `title` correcto
- `h1` visible
- botón `Comparte tu link` presente
- click con `navigator.share` stubbeado deja feedback visible:
  - `Listo. Se abrió el menú para compartir su link.`

## Resultado

La ruta nueva ya está lista como superficie pública aislada para el programa de referidos y puede enlazarse desde campañas o WhatsApp.
