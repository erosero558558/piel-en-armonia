## Implemented

1. Added availability productivity action in admin UI:
- New button: `data-action="availability-next-with-slots"` in calendar header.
- New inline shortcut hint for availability workflow.
- Bumped admin cache-buster to `admin-20260226-nav5`.

2. Availability module hardening:
- Persisted last selected date in local storage (`admin-availability-last-selected-date`).
- Restored selected date on availability init when valid.
- Added jump helper to next date with configured slots.
- Added keyboard day-to-day navigation in calendar cells (`ArrowLeft/Right/Up/Down`).
- Added helper exports:
  - `jumpAvailabilityToNextWithSlots()`
  - `focusAvailabilityTimeInput()`
  - `isAvailabilitySectionActive()`

3. Admin keyboard/action wiring:
- Added `/` shortcut in availability section to focus time input.
- Added availability shortcuts:
  - `Alt+Shift+Y`: today
  - `Alt+Shift+S`: next date with slots
  - `Alt+Shift+ArrowLeft/ArrowRight`: month navigation
  - `Alt+Shift+D`: duplicate selected day to next day
  - `Alt+Shift+V`: paste day slots
  - `Alt+Shift+X`: clear selected day
- Added click handler for `availability-next-with-slots`.
