## Objetivo

Auditar [images/optimized](/Users/luciaguadalupecaizasanchez/Documents/GitHub/Aurora-Derm/images/optimized) para verificar que `.webp` siguen referenciadas desde HTML/CSS/superficies web vivas y listar las huerfanas sin borrarlas.

## Comandos usados

```bash
find images/optimized -type f | wc -l
find images/optimized -type f | awk -F/ '{print $NF}' | sed 's/.*\\.//' | sort | uniq -c
rg -n "images/optimized|/images/optimized" -g '*.html' -g '*.php' -g '*.css' -g '*.js' -g '*.astro' -g '*.ts' -g '*.tsx' .
node /tmp/s4-25-images-audit-fast.js
```
