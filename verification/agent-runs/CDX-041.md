# CDX-041 - Release pipeline para apps Turnero

## Objetivo

Construir y publicar un pipeline reproducible para:

- `Turnero Operador`
- `Turnero Kiosco`
- `Turnero Sala TV`

sin mezclar los binarios con el deploy general del sitio y dejando listas las rutas que ya consume `admin.html#queue` y `/app-downloads/`.

## Alcance

- Workflow manual `.github/workflows/release-turnero-apps.yml`
- Empaquetador `bin/stage-turnero-app-release.js`
- Feed de auto-update para desktop en `desktop-updates/stable/...`
- Manifest canonico en `app-downloads/stable/release-manifest.json`
- Lectura runtime del manifest desde `lib/AppDownloadsCatalog.php`
- Versionado release para Android TV y desktop
- Contratos Node del workflow y del empaquetador

## Checklist

- [x] Build Windows y macOS para `Operador`
- [x] Build Windows y macOS para `Kiosco`
- [x] Build Android TV para `Sala TV`
- [x] Bundle final con `app-downloads/` y `desktop-updates/`
- [x] Upload como artifact de GitHub Actions
- [x] Publicacion opcional al hosting con `FTP_*`
- [x] Publicacion opcional a GitHub Release
- [x] Manifest de release para que el admin lea version/fecha reales
- [x] Tests de contrato agregados a `agent:test`

## Validacion prevista

- `node --test tests-node/release-turnero-apps-workflow-contract.test.js tests-node/stage-turnero-app-release-script.test.js`
- `npm run agent:test`
- Workflow manual `Release Turnero Apps` en modo `dry_run=true`

## Limitaciones de esta terminal

- No pude correr `node` ni `npm` localmente porque no estan instalados en esta sesion.
- No pude correr `php -l` ni validar el merge runtime del catalogo porque `php` tampoco esta disponible.
- No pude generar `.exe`, `.dmg` o `.apk` locales; el flujo real queda delegado al workflow nuevo.

## Riesgos abiertos

- `src/apps/turnero-desktop` sigue sin `package-lock.json`, por lo que el workflow usa `npm install` y no `npm ci`.
- La publicacion a hosting depende de `FTP_SERVER`, `FTP_USERNAME` y `FTP_PASSWORD`.
- macOS y Windows siguen sin firma/notarizacion en esta primera ola.
