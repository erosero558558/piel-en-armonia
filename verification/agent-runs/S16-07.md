# S16-07 — Smoke de entrega Papertrail

## Resumen

- Se fortalecio `lib/logger.php` para que el fallback sin Monolog use `stderr` explicito y pueda enviar UDP con formato syslog minimo cuando `PAPERTRAIL_HOST` y `PAPERTRAIL_PORT` estan configurados.
- Se reescribio `tests/Smoke/PapertrailSmokeTest.php` para validar en subprocesos aislados:
  - fallback a `stderr` cuando no hay destino Papertrail
  - entrega UDP con `PRI`, timestamp, hostname, canal y payload JSON con el mensaje smoke

## Archivos

- `lib/logger.php`
- `tests/Smoke/PapertrailSmokeTest.php`

## Verificacion

```bash
php -l lib/logger.php
php -l tests/Smoke/PapertrailSmokeTest.php
php tests/Smoke/PapertrailSmokeTest.php
```
