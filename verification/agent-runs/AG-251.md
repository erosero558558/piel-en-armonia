# AG-251 - Operator auth runtime

## Resumen

Se endurecio el diagnostico de `operator_auth` para distinguir:

- `auth_status_http_530`: el endpoint canonico falla por HTTP sin caer en
  `mode mismatch`
- `auth_edge_failure`: tanto `operator-auth-status` como la fachada
  `admin-auth.php?action=status` fallan por edge/origen
- `facade_only_rollout`: la fachada ya publica contrato OpenClaw, pero el
  surface canonico aun no

Ademas, el runtime transversal quedo documentado con esas nuevas
clasificaciones.
