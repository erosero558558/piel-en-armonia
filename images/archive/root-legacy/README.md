# Root Legacy Image Archive

Este directorio preserva imagenes que alguna vez vivieron en la raiz del repo
pero ya no forman parte del runtime publico/admin activo.

Reglas:

- Se conservan solo como referencia historica.
- No son parte del gateway publico V6 ni del shell admin.
- Si un asset visual vuelve a ser necesario, debe salir desde una ruta canonica
  (`images/optimized/**`, `images/src/**` u otra superficie documentada), no
  desde la raiz por inercia.
