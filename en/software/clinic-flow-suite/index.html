    FAQ

    What clinics ask before buying

    Do we need proprietary hardware to launch?

    No. The pilot is designed to work with a browser, a TV or monitor, a PC or tablet, and a compatible thermal printer.

    Does this replace the clinic's full medical software?

    Not at this stage. The suite competes as waiting room operations + booking + reminders + reporting, not as a full EMR.

    Can we start with a single site?

    Yes. The first model is built for a pilot site, then expands to branding, lite multi-site, and dashboard layers only if the flow proves itself.

    How do you reduce no-show if patients never touch the kiosk?

    The improvement does not depend on the kiosk alone. It comes from reminders, mobile check-in, queue visibility, and a front desk that sees the same state.

    If reception adopts it quickly, patients understand it, and management can measure it, the suite already has a buying story.

    Start with daily operations, prove the patient flow, and close with the site dashboard when the budget conversation shows up.
