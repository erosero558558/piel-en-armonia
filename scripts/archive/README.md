# Archived Root Scripts

Este directorio agrupa scripts legacy y utilitarios one-off movidos fuera de la
raiz para reducir ruido operativo.

Reglas:

- Se preservan solo para referencia manual o forense.
- No forman parte del carril diario recomendado.
- Para comandos activos preferir `package.json`, `bin/**` y
  `docs/OPERATIONS_INDEX.md`.
- Incluye runners retirados del admin legacy/v2, como `run-admin-queue-v2.js`.
- Incluye tombstones ejecutables retirados, como `jules-dispatch.js` y
  `kimi-run.js`.
